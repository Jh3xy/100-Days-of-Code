This utilizes links to external documents, sections in the same document

Here are three types of links;

  • External Link: that links to a website outside of the current site. Example; Link to external site
  • Internal that links to a section in the same html document. Example; About
  • Relative link that links to another document in the same folder. Example;Dummy Site!
